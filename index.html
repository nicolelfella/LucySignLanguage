LUCY - Sign Language Tool

Goal

Develop an algorithm to use the Myo Gesture Control Armband to translate sign language into text and speech. As part of our proof of concept, we are working on mapping specific hand gestures to encoded signals that later on can be used to develop a translation.

Technical Details

Using the Myo SDK to read electromyography (EMG) signals from the wristband
